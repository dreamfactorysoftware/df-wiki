# DreamFactory Wiki
> https://wiki.dreamfactory.com

DreamFactory is an open-source API generation and management platform. It automatically generates REST APIs for any database, handles authentication, role-based access control, API rate limiting, and much more.

This wiki is the single source of truth for all DreamFactory documentation.
